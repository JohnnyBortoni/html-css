Sêmantica no HTML

No HTMl5 eles focaram em sêmantica (significado) para as tags, para que os navegadores consiguam entender melhor. Ao invés de só focar na forma, como o antigo HTML fazia.
Exemplo:

A tag < font > ficou obsleta, pelo fato que se pode usar o CSS para mudar o tipo da fonte, tamnho e cor

A tag < big > e < small > ficaram obsletas também por causa que se pode usar o CSS para mudar o tamanho do texto

A tag < b > foi substítuida pela tag < strong >


Listas

Listas ordenadas:

Usando a tag < ol > mais < li > pode se criar listas organizadas por números (1), alfabeto minusculo (a) ou maiusculo (A) e número romanos minusculo (i) ou maiusculo (I), usando o paranmetro type. E se vôce quiser começar em número sem ser o 1 se pode colocar o parâmetro start e o número desejado.

Lista não ordenada

Usando a tag < ul > mais < li >, pode se criar listas organizadas com bolas pretas (disc), quadrados (square) e círculos (circle). Usando o parâmetro type.
Exemplo:

  • Pão
  • Tomate
  • Azeite
  • Pão
  • Tomate
  • Azeite
  • Pão
  • Tomate
  • Azeite
